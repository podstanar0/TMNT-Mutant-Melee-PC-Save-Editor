# TMNT Mutant Melee PC Save Editor
SaveGame editor for TMNT Mutant Melee PC game version, written in C#.

**WARNING**: spaghetti code ahead.

## Usage:
Hit **Load**, look for `%gameDir%/tmnt.sv` file, open it, and after editing hit **Save**.
